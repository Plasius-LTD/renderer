# Renderer Public Package Scope

## Goal

Standardize `@plasius/renderer` as a standalone public package with the same governance, documentation, and validation expectations used by other Plasius package repositories.

## In Scope

- public package docs and governance assets
- contributor guidance and CLA references
- public-package validation and packaging checks
- repository-level CI and release readiness

## Out of Scope

- redesigning the renderer API
- changing the underlying rendering stack
- introducing Player System-specific runtime behaviour into the renderer package
